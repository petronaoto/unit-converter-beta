Theory & Specifications Manual

Complete technical reference for all formulas, standards, coefficients and algorithmic logic implemented in v2.2. Primary reference: JIS K 2301:2011 and ISO 6578:1991.

Part I — Gas Compositional Analysis (JIS K 2301:2011)

1.1 Component Physical Constants (JIS K 2301 Table)

1.2 Volumetric → Mole Fraction Conversion (JIS K 2301 §7)

Step 1
Cmi = Cvᵢ / Zᵢ   (un-normalized mol proportional)
Cvᵢ = volumetric fraction of component i; Zᵢ = pure-component compressibility at standard conditions
Step 2
Cmᵢ = ROUND( Cmi / Σ Cm , 4 )   [Cmi % — rounded to 4 d.p.]
Cmᵢ = normalized mole fraction. Rounding to 4 d.p. is mandated by the Excel reference (JIS K 2301 calculation worksheet)
Reverse
Cvᵢ (recovered) = ( Cmᵢ × Zᵢ ) / Σ(Cm × Zₖ)   [Cvᵢ]
Mol% input mode: user enters Cmᵢ → displayed vol fracs follow the Cvᵢ (recovered) formula

1.3 Sample Gas Compressibility Factor (Z)

Cmᵢ × √bᵢ = ROUND( Cmᵢ × √bᵢ , 5 )   [rounded per-component to 5 d.p.]
Z = 1 − (Σ Cmᵢ × √bᵢ)²
Z_rounded = ROUND(Z, 4)   [used for FLOW density calculation]
Example (CH₄=89%, C₂=7%, C₃=2.5%, iC₄=0.7%, nC₄=0.5%, N₂=0.3%):
Σ Cm√b = 0.8887×0.049 + 0.0704×0.1 + 0.0254×0.1453 + ... = 0.05693
Z = 1 − 0.05693² = 0.9968 (4 d.p.)
⚡ CRITICAL: Z_exact (before rounding) is used for HHV/SG calculations; Z_rounded (4 d.p.) is used for standard gas density ρ_std. This matches the Excel-based Calc. sheet exactly.

1.4 Gross Calorific Value / HHV

H = ROUND( Σ(Cmᵢ × Hᵢ) / Z_exact , 2 ) [MJ/Nm³]
No per-component rounding of Cmᵢ×Hᵢ before summing. Z_exact = 1−(ΣCm√b)². Final result rounded to 2 d.p. → 1.4 Gross Calorific Value / HHV.
Reference example:
Σ(Cmᵢ×Hᵢ) = 0.8887×39.84 + 0.0704×69.79 + ... = 44.444
H = 44.444 / 0.9968 = 44.587... → 44.59 MJ/Nm³

1.5 Specific Gravity & Wobbe Index

S = ROUND( Σ(Cmᵢ × Sᵢ) / Z_exact , 3 ) [Air=1]
WI = ROUND( H₂ dp / √(S₃ dp) , 2 ) [MJ/Nm³]
Sᵢ = specific gravity of pure component i relative to air. WI uses the already-rounded H (2 d.p.) and S (3 d.p.) — not the exact intermediate values. This is critical for matching Wobbe Index = 56.00.
S = 0.6321 / 0.9968 = 0.634 (3 d.p.)
WI = 44.59 / √0.634 = 55.997... → 56.00

1.6 Maximum Combustion Point (MCP)

Base MCP = ROUND( Σ(Aᵢ × fᵢ × SFᵢ) / Σ(Aᵢ × fᵢ) , 2 )
Aᵢ = vol% of combustible component i; fᵢ = combustibility factor; SFᵢ = flame speed (cm/s)
K correction (inert dilution):
denom = 100 − 4.77 × pO₂
K = (ΣAᵢ/ΣAᵢaᵢ) × [ (2.5·pCO₂ + pN₂ − 3.77·pO₂)/denom + ((pN₂−3.77·pO₂)/denom)² ]
MCP = ROUND( Base_MCP × (1 − K) , 1 )

1.7 Lower Heating Value (LHV) — 真発熱量 (JIS K 2301 Table 30)

LHV = ROUND( Σ(Cmᵢ × LHVᵢ) / Z_exact , 2 ) [MJ/Nm³]
Formula mirrors HHV (§1.4). LHVᵢ is the net calorific value of pure component i at standard conditions (0°C, 101.325 kPa, gaseous water as product) from JIS K 2301:2011 Table 30. Same Z_exact denominator and 2 d.p. rounding apply.
LHVᵢ = HHVᵢ − n_H₂O × ΔHvap(H₂O,0°C) / Vm
ΔHvap correction ≈ 2.00 MJ per Nm³ per mol H₂O (H₂: 1 mol H₂O; CH₄: 2; C₂H₆: 3; etc.)
⚠ Wobbe Index always uses HHV regardless of mode — per JIS K 2301 §7 definition.
⚠ All LHV values from JIS K 2301:2011 Table 30. ΔHvap(H₂O) implied = 2.011 MJ/Nm³ per mol H₂O at 0°C (anchored on CH₄ = 35.818 MJ/Nm³).
Reference example (same composition as §1.4):
Σ(Cmᵢ×LHVᵢ) = 0.8887×35.818 + 0.0704×63.76 + 0.0254×91.18 + ... = 40.116...
LHV = 40.116 / 0.9968 = 40.246... → 40.25 MJ/Nm³

Part II — Standard Gas Density & Flow Conversions

2.1 Standard Gas Density ρ_std

ρ_std = P·(MW/1000) / (Z_rounded·R·T_std) [kg/Nm³]
P = 101325 Pa (1 atm, standard)
MW = Σ(Cmᵢ × MWᵢ) [g/mol], where Cmᵢ is rounded to 4 d.p.
Z_rounded = ROUND(1 − (ΣCm√b)², 4)  ← ROUNDED Z used here for exact Excel match
R = 8.31446262 J/(mol·K); T_std = 273.15 K (0°C)
Reference: CH₄=89%, C₂=7%, C₃=2.5%, iC₄=0.7%, nC₄=0.5%, N₂=0.3%
MW = 18.3049839 g/mol, Z_rounded = 0.9968
ρ_std = 101325 × (18.3049839/1000) / (0.9968 × 8.31446262 × 273.15) = 0.81930 kg/Nm³

2.2 Mass ↔ Volume Flow Conversion

Mass → Volume (standard):
Q_std = ṁ / ρ_std [Nm³/h]
Output units: kNm³/h (÷1000), kNm³/d (÷1000×24), mscfd
100 ton/h = 100,000 kg/h
→ 100,000 / 0.81930 / 1000 = 122.056 kNm³/h
Volume → Mass (standard):
ṁ = Q_std × ρ_std [kg/h]
Input units: kNm³/h (×1000), kNm³/d (×1000÷24), mscfd
100 kNm³/h = 100,000 m³/h (Nm³)
→ 100,000 × 0.81930 / 1000 = 81.930 ton/h
Actual Conditions (non-standard T, P):
Q_act = Q_std × (T_act/T_std) × (P_std/P_act)
Applies ideal gas temperature/pressure correction. When T≠0°C or P≠0 gauge, output units switch from kNm³/h to kAm³/h.

Part III — LNG Liquid Density (ISO 6578:1991 Klosek-McKinley)

3.1 Klosek-McKinley Method Overview

The ISO 6578 Klosek-McKinley method computes LNG liquid density by summing the molar volumes of pure-component liquids, then applying correction factors k₁ and k₂ for CH₄ and N₂ interactions.

V_mix = Σ(xᵢ · Vmᵢ(T)) − x(CH₄)·k₁(MW,T) − x(N₂)·k₂(MW,T)
ρ_LNG = MW_mix / V_mix [kg/m³]
xᵢ = mole fraction; Vmᵢ(T) = molar volume at temperature T [m³/kmol]; k₁, k₂ = correction factors from ISO 6578 Table C, interpolated by MW_mix

Part IV — Hydraulics & Gas Laws (Advanced Tab)

4.1 Z-Factor: Hall-Yarborough Method

Pseudo-reduced vars: T_pr = T/Tc   P_pr = P/Pc
t = 1/T_pr   (dimensionless)
A = −0.06125·t·exp(−1.2·(1−t)²)
Z computed by iterative solution of the Hall-Yarborough equation in reduced density y
Accurate for P_pr < 15, T_pr > 1.2

4.2 Pressure Drop: Darcy-Weisbach + Colebrook-White Friction Factor

ΔP = f·(L/D)·(ρ·v²/2) + ρ·g·Δz   [Pa]
Iterative Colebrook-White equation — turbulent regime (Re ≥ 2300):
1/√f = −2·log₁₀( ε/(3.7·D) + 2.51/(Re·√f) )
Laminar regime (Re < 2300): f = 64/Re
Solved iteratively to convergence  (tolerance = 1×10⁻⁶, max 100 iterations)
ε = pipe absolute roughness [m]; default 0.045 mm (commercial steel)

Part VI — PRV Sizing Theory (API 520 §5.6–§5.10)

6.1 Gas/Vapor — Critical Flow (§5.6.3, Eq. 2/5)

Critical check: P2 ≤ Pcf = P1 × (2/(k+1))^(k/(k−1))  [Eq. 1]
A = W / (C · Kd · P1 · Kb · Kc) × √(T·Z/M)
C (USC) = 520 × √( k · (2/(k+1))^((k+1)/(k−1)) )  [Eq. 8]
C (SI) = 0.03948 × √( k · (2/(k+1))^((k+1)/(k−1)) )  [Eq. 9]
Kd = 0.975 (PRV) · Kb = 1.0 (conventional/pilot) · Kc = 1.0 (no rupture disk)

6.2 Gas/Vapor — Subcritical Flow (§5.6.4, Eq. 12/15)

F2 = √[ k/(k−1) · r^(2/k) · (1−r^((k−1)/k)) / (1−r) ]   r = P2/P1
USC: A = W / (735 · F2 · Kd · Kc) × √(T·Z / (M·P1·(P1−P2)))
SI: A = 17.9·W / (F2 · Kd · Kc) × √(T·Z / (M·P1·(P1−P2)))

6.3 Steam (§5.7, Eq. 21/22)

USC: A = W / (51.5 · P1 · Kd · Kb · Kc · KN · KSH)
SI: A = 190.5·W / (P1 · Kd · Kb · Kc · KN · KSH)
KN = 1.0 (P1 ≤ 1500 psia); KN = (0.1906·P1−1000)/(0.2292·P1−1061) (1500 < P1 ≤ 3200 psia)
KSH = 1.0 for saturated steam; from Table 9 (API 520) for superheated steam

6.4 Liquid — Certified PRV (§5.8, Eq. 28/29)

USC: A = Q / (38 · Kd · Kw · Kc · Kv) × √(Gl / (P1−P2))
SI: A = 11.78·Q / (Kd · Kw · Kc · Kv) × √(Gl / (P1−P2))
Kv = (0.9935 + 2.878/√Re + 342.75/Re^1.5)^(−1)  [Eq. 30]
Re = Q·(2800·Gl)/(µ·√A) [USC] or Q·(18800·Gl)/(µ·√A) [SI]
Iterative: size with Kv=1 → Re from next API 526 size → Kv → resize

6.5 Liquid — Non-Certified PRV (§5.9, Eq. 38/39)

USC: A = Q / (38 · Kd · Kw · Kc · Kv · Kp) × √(Gl / (1.25·Ps−P2))
Kp = 1.0 at 25% overpressure; Kp ≈ 0.6 at 10% overpressure (Fig. 38). Default Kd = 0.62.

6.6 Two-Phase — Omega Method (§5.10 / Annex C.2.2)

Step 1: ω = 9·(v9/vo − 1)  [Eq. C.12]   v9 = spec. vol. at isenthalpic flash to 0.9·Po
Step 2: η_c ≈ [1 + (1.0446−0.0093431·√ω)·ω^(−0.56261)]^(−0.70356+0.014685·lnω)  [Eq. C.15]
Step 3 (critical): G = 68.09·η_c·√(Po/(vo·ω)) [USC] | G = η_c·√(Po/(vo·ω)) [SI, Po in Pa]
A = 0.04·W / (Kd·Kb·Kc·Kv·G) [USC, in²]  |  A = 277.8·W / (Kd·Kb·Kc·Kv·G) [SI, mm²]
